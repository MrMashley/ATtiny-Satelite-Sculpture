# ATtiny-Satellite-Sculpture


ATtiny85 based satelite sculptur inspired by the work of [Mohit Bhoite](https://bhoite.com/sculptures/)


![IMG_8489](https://github.com/user-attachments/assets/a5e831b3-cd72-4594-9e55-7aa5f980587c)


Solar cells charge a capacitor which provides power to the attiny85

The ATTiny goes into sleep mode to conserve power and wakes every 16 (or so) seconds, reads the voltage from the solar panels and blink an LED if the voltage is below a set threashold. 


Parts List

- Solar cells: SM141K04LV
- Diode: 1N4148
- Capacitor: CXP-3R0105R-TWV
- ATtiny: ATTINY85-20PU-ND
- LED: Diffused 3mm yellow LED
- 0.8mm Brass Rod
- Bell jar from IKEA
